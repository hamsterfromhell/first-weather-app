# 🌤️ Weather App

A live weather app built with **HTML**, **CSS** and **JavaScript** that fetches real-time weather data using the OpenWeatherMap API.

![Weather App Preview](<img width="1280" height="712" alt="preview" src="https://github.com/user-attachments/assets/91f8d1f3-b084-49d7-b4cf-23d48259e099" />
)

## ✨ Features

- 🔍 Search weather by city or country
- 🌡️ Displays live temperature in °C
- 💧 Shows humidity percentage
- 💨 Shows wind speed
- 🌧️ Dynamic weather icons based on conditions
- ❌ Error handling for invalid city names

## 🛠️ Built With

- HTML5
- CSS3
- Vanilla JavaScript
- [OpenWeatherMap API](https://openweathermap.org/api)

## 🚀 How to Use

1. Type a city or country name in the search box
2. Click the search button
3. See live weather data instantly!
